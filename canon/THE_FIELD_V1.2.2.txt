4. Object Ontology

An OBJECT is defined as the coexistence of four autonomous layers:
	•	LO — Logical Object
	•	SO — Symbolic Object
	•	GO — Geometric Object
	•	BO — Behavioral Object

Lifecycle rule:
Layers may be constructed in any order,
but no layer may infer, overwrite, validate, or complete another.

⸻

5. Object Layers (Authoritative Definitions)

5.1 LO — Logical Object
	•	Domain: semantic
	•	Produced by: SYSTEM3

Contents:
	•	invariants (what cannot change)
	•	constraints (what may vary)
	•	relational rules (allowed relations)
	•	exclusion rules (forbidden states)

Explicit exclusions:
	•	geometry
	•	materials
	•	scale

LO never references physical form.


5.2 SO — Symbolic Object
	•	Domain: symbolic

Contents:
	•	identity (name, place, date, architect)
	•	cultural framing
	•	historical context
	•	documentary sources

Certitude constraints:
	•	SO has no causal power
	•	SO cannot justify geometry
	•	SO cannot validate behavior

SO expresses meaning only.


5.3 GO — Geometric Object
	•	Domain: geometric
	•	Authority: explicit only

Allowed sources:
	•	measured plans
	•	measured sections
	•	surveyed drawings
	•	photographs with declared anchor authority

Contents:
	•	dimensions (length, width, height)
	•	elements (walls, slabs, openings)
	•	relations (adjacency, alignment, curvature)
	•	material extents (surface, thickness)

Confidence states (mandatory per element):
	•	measured
	•	anchored (image-based)
	•	inferred (explicitly flagged)
	•	unresolved

A GO may contain mixed confidence states.
Each geometric element must declare one.

Prohibitions:
	•	inferred dimensions without flag
	•	semantic guessing
	•	render-based geometry recovery


5.4 BO — Behavioral Object
	•	Domain: behavioral

Describes:
	•	circulation tendencies
	•	spatial affordances
	•	perceptual sequences
	•	temporal use patterns

Certitude constraints:
	•	not intent
	•	not symbolism
	•	not prediction
	•	not optimization

BO describes what space affords, not what people do.

# Public Package Scope

This package preserves the usable package layer from the richer release while replacing
the canon with verbatim author-provided public canonical texts.

## Included

- boot flow
- public canon
- ontology layer (THE_FIELD)
- examples
- operational templates

## Excluded

- deeper internal SYSTEM3 core architecture
- private engine and routing internals
- research-sensitive internal semantic layers
- `SYSTEM03_CANONICAL.txt`

## Design principle

Canon is included verbatim.
Docs and templates support usage.
Private engine internals remain outside this public package.
